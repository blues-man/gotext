Ciao. This is goText.

This page is here to celebrate goText, an opensource software used some years ago to send free sms worldwide from Mobile, PDA, Desktops available on Sourceforge and released under the GNU GPL V2.

Learn about goText

How it works

goText was initially made as J2ME app to avoid or reduce cost of SMS sending, creating an open source SMS Gateway to consume multiple free and paid Services across the world.

SMS

goText is mainly a small Java software for mobile phones that you can use to send free SMS messages to anyone in the world. The software itself is free, the download too, and the same applies to SMS messages
